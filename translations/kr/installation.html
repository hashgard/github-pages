# Hashgard 설치하기

이 가이드는 hashgard와 hashgardcli를 엔트리포인트를 시스템에 설치하는 방법을 설명합니다. hashgard와 hashgardcli가 설치된 서버를 통해 풀노드 또는 밸리데이터로써 최신 테스트넷에 참가하실 수 있습니다.

# Go 설치하기

공식 Go 문서를 따라서 go를 설치하십시오. $GOPATH, 그리고 $PATH의 환경을 꼭 세팅하세요. 예시:

mkdir -p $HOME/go/bin
echo "export GOPATH=$HOME/go" >> ~/.bash_profile
echo "export PATH=\$PATH:\$GOPATH/bin" >> ~/.bash_profile
echo "export GO111MODULE=on" >> ~/.bash_profile
source ~/.bash_profile

TIP

코스모스 SDK를 운영하기 위해서는 Go 1.11.ㅎ+ 이상 버전이 필요합니다.

# 바이너리 설치하기

다음은 최신 Hashgard 버전을 설치하는 것입니다. 예시에서는 최신 스테이블 릴리즈가 포함되어있는 master 브랜치를 이용해 진행됩니다. 필요에 따라 git checkout을 통해 최신 릴리즈가 설치되어있는지 확인하세요.

git clone -b <latest-release-tag> https://gitlab.hashgard.com/hashgard/hashgard
cd gaia && make install

참고: 여기에서 문제가 발생한다면, Go의 최신 스테이블 버전이 설치되어있는지 확인하십시오.

위 절차를 따라하시면 hashgard와 hashgardcli 바이너리가 설치될 것입니다. 설치가 잘 되어있는지 확인하십시오:

$ hashgard version
$ hashgardcli version
